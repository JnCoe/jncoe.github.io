## üé® How to Use This Project

If you'd like to use this website as a starting point for your own:

1. **Fork the repository** - Create your own copy
2. **Customize the design** - Make it visually unique! Don't create an exact clone
3. **Replace all content** - Add your own bio, blog posts, projects, and information
4. **Keep the license** - Maintain the GPLv3 license file and attribution comments
5. **Share improvements** - Consider submitting pull requests for bug fixes or enhancements

### Example: Customization Ideas

- Change the color scheme in `css/variables.css`
- Modify layouts in `src/layouts/` and `src/components/`
- Adjust typography and spacing
- Reorganize sections
- Add new features (e.g., dark mode toggle, different theme options)

## ‚úçÔ∏è Code Style Guide

To maintain consistency:

- **JavaScript**: Use ES6+ syntax, add JSDoc comments for functions

- **File naming**: Use kebab-case for files, camelCase for variables
